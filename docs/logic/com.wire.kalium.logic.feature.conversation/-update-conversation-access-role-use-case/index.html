UpdateConversationAccessRoleUseCase

class UpdateConversationAccessRoleUseCase(conversationRepository: ConversationRepository)

This use case will update the access role configuration of a conversation. So we can operate and allow or not operation based on these roles:

  • Conversation.isGuestAllowed

  • Conversation.isServicesAllowed

  • Conversation.isTeamGroup

  • Conversation.isNonTeamMemberAllowed
