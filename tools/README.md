# Tools

Development and build-time utilities for code generation and testing.

**Modules:**
- `protobuf-codegen` - Protocol buffer code generation
- `testservice` - Test service infrastructure for QA and integration testing
- `backup-verification` - Backup file verification and validation tools
- `monkeys` - Automated testing and chaos engineering tools

**Dependencies:** Can depend on any module as needed for tooling purposes

**Used by:** Build process and development workflows

**Guidelines:** Tools are not included in production artifacts. Use for development only.
